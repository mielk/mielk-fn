				function isObject(value: unknown): boolean

Parameters

value: unknown
The value to be checked for being an object.

Returns

: .

Examples

				// Example 1:
				const obj = { key: 'value' };
				console.log(isObject(obj)); // Output: true

				const arr = [1, 2, 3];

				// Example 3:
