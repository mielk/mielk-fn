isObject

Summary

Determines if the given value is an object, excluding arrays and null values.

Signature

function isObject(value: unknown): boolean

Parameters

value: unknown
The value to be checked for being an object.

Returns

boolean: Returns true if the input value is an object (excluding arrays and null values), and false otherwise.

Examples

            // Example 1:
            const obj = { key: 'value' };
            console.log(isObject(obj)); // Output: true

            // Example 2:
            const arr = [1, 2, 3];
            console.log(isObject(arr)); // Output: false

            // Example 3:
            console.log(isObject(null)); // Output: false
